# MAML Skill: Self-Healing Error Logger

**Skill Name:** `self-healing-error-logger`  
**Version:** 1.0.0  
**Type:** `skill`  
**Target Harnesses:** Local Harness or Agent 
**Purpose:** Detect failures, generate recovery MAML logs, suggest fixes, and append detailed entries to History for self-improvement.

---

## Intent

Automatically handle execution errors by creating structured recovery logs and suggesting corrective actions, enabling the agent to learn from failures.

---

## Context

This skill turns errors into learning opportunities. It generates a new recovery `.maml.md` file and enriches the current file’s History. Perfect for robust, long-running agent systems.

---

## Code_Blocks

```python
import yaml
from datetime import datetime
from typing import Dict, Any

def handle_error(error: Exception, original_maml: str, context: Dict = None) -> Dict[str, Any]:
    """
    Main self-healing entry point.
    Creates recovery MAML and logs details.
    """
    timestamp = datetime.now().isoformat()
    
    # Analyze error
    error_type = type(error).__name__
    suggested_fix = generate_suggested_fix(error_type, str(error))
    
    # Create recovery MAML
    recovery_maml = create_recovery_maml(original_maml, error, suggested_fix, timestamp)
    
    recovery_path = f"recovery-{timestamp[:19].replace(':', '-')}.maml.md"
    with open(recovery_path, "w") as f:
        f.write(recovery_maml)
    
    # Return structured result
    return {
        "status": "recovery_initiated",
        "error_type": error_type,
        "error_message": str(error),
        "suggested_fix": suggested_fix,
        "recovery_file": recovery_path,
        "timestamp": timestamp
    }

def generate_suggested_fix(error_type: str, message: str) -> str:
    """Generate context-aware fix suggestions"""
    if "ValidationError" in error_type:
        return "Check Input_Schema compliance and data types."
    elif "FileNotFound" in error_type:
        return "Verify file paths in Context and use absolute paths."
    elif "API" in message:
        return "Check API key, network connectivity, and retry with exponential backoff."
    return "Review code blocks for edge cases and add more robust error handling."

def create_recovery_maml(original_maml: str, error: Exception, suggested_fix: str, timestamp: str) -> str:
    """Generate a new recovery MAML file content"""
    recovery = f"""---
maml_version: "1.0.0"
id: "urn:uuid:recovery-{timestamp[:19]}"
type: "error_recovery"
origin: "agent://self-healing-logger"
created_at: "{timestamp}"
---

## Intent
Recovery from previous execution failure.

## Context
Original error: {str(error)}

## Recovery_Steps
- {suggested_fix}
- Review History for patterns
- Re-execute with corrected parameters

## History
- {timestamp}: [ERROR] {type(error).__name__}: {str(error)}
- {timestamp}: [RECOVERY] Generated by self-healing logger
"""
    return recovery
```
