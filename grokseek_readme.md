## üöÄ **What is GROKSEEK?**

**GROKSEEK OS** is a **single HTML file operating system** that turns any device into a **Bluetooth-powered AI gateway hub**.  
It fuses **Grok (xAI)** and **DeepSeek** into a **hybrid AI agent**, accessible via a **pixel-perfect, Windows 95-inspired UI** with **8-bit aesthetics**.

- **No internet required** after setup  
- **Bluetooth Mesh Networking** (ESP32/Arduino compatible)  
- **Local file system simulation**  
- **API popup wizard** for Grok & DeepSeek  
- **Git push/pull directly from OS**  
- **Runs on Netlify, GitHub Pages, or locally**
