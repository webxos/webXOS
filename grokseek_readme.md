## üöÄ **What is GROKSEEK?**

**GROKSEEK OS** is a **single HTML file operating system** that turns any device into a **Bluetooth-powered AI gateway hub**.  
It fuses **Grok (xAI)** and **DeepSeek** into a **hybrid AI agent**, accessible via a **pixel-perfect, Windows 95-inspired UI** with **8-bit aesthetics**.

- **No internet required** after setup  
- **Bluetooth Mesh Networking** (ESP32/Arduino compatible)  
- **Local file system simulation**  
- **API popup wizard** for Grok & DeepSeek  
- **Git push/pull directly from OS**  
- **Runs on Netlify, GitHub Pages, or locally**

## ‚ö° **Features**

| Feature | Description |
|--------|-------------|
| **Bluetooth Mesh OS** | Connect ESP32, Arduino, phones, sensors via BLE mesh |
| **Hybrid AI Agent** | GROK + DEEPSEEK unified terminal (`grokseek >`) |
| **API Wizard** | Popup guide to connect Grok/DeepSeek APIs |
| **Git Integration** | Push/pull directly from OS (GitHub, GitLab) |
| **8-bit Pixel UI** | Retro Windows 95 look with neon glow |
| **Local File System** | Simulated FS for configs, logs, scripts |
| **Zero Install** | One `index.html` ‚Äì run anywhere |
| **Admin Login** | Secure boot with password (configurable) |
| **Offline-First** | Works without internet after API setup |
