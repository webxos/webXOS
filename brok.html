‚öôÔ∏è Backend Installation Guide

This is a FRONTEND ONLY demo page. To run the full agentic firewall, you must install and deploy the Python backend.

Prerequisites

  • Python 3.8 or higher
  • Git installed on your system
  • Basic understanding of command line operations
  • Admin/root access for network operations

Option 1: Deploy on GitHub

STEP 1: Create a new repository on GitHub
STEP 2: Clone your repository
git clone https://github.com/yourusername/brok-firewall.git cd brok-firewall
STEP 3: Create the Python file
# Create brok11.py with the provided Python code nano brok11.py # Paste the Python code and save (Ctrl+X, Y, Enter)
STEP 4: Create requirements.txt
echo "# No external dependencies required" > requirements.txt # Brok uses only Python standard library
STEP 5: Push to GitHub
git add . git commit -m "Initial Brok Agentic Firewall" git push origin main

Option 2: Run Locally

STEP 1: Create a project directory
mkdir brok-firewall cd brok-firewall
STEP 2: Save the Python code as brok11.py
STEP 3: Run the firewall
python3 brok11.py # Or with sudo for enhanced network access sudo python3 brok11.py
The system will initialize 100 agents and begin monitoring your network at 10Hz frequency. Press Ctrl+C to safely shutdown.

Option 3: Deploy on Netlify (Hybrid Approach)

Netlify is designed for static sites and serverless functions. For the full Brok system:

STEP 1: Deploy this frontend to Netlify
# Install Netlify CLI npm install -g netlify-cli # Login to Netlify netlify login # Deploy netlify deploy --prod
STEP 2: Run backend on a VPS or cloud server

The Python backend requires persistent network access and should be deployed on:

  • AWS EC2 / Lightsail
  • Google Cloud Compute Engine
  • DigitalOcean Droplet
  • Your own server/VPS
STEP 3: Create API endpoints (optional)
# Use Flask or FastAPI to create REST endpoints pip install flask flask-cors # Wrap Brok functions in API routes for frontend communication
