ABOUT GINUX

GINUX is an upcoming gaming platform under development for Linux OS. It represents a global hub for Linux gamers, bridging the gap between web interfaces and local game libraries.

The public website triggers a local bridge via custom URL protocol for seamless game launches directly from your browser. No more fumbling with multiple launchers or complicated setup processes.

Built with the Linux gaming community in mind, GINUX combines retro aesthetics with modern functionality to create a unique gaming experience.

KEY FEATURES WILL INCLUDE

One-Click Launches

GINUX will Launch games directly from your browser with a single click using custom URL protocols.

Wine Integration

GINUX will offer seamless support for Windows games via Wine, including Battle.net and other platforms.

Open Source Bridge

GINUX will include a Lightweight local bridge script that securely connects web interface to your local games.

8-bit Retro UI design

Modern sci-fi interface with black darkscale background, neon green matrix glow, and neural accents.

Web-Based Interface

Access your game library from any device with a web browser. Your local bridge handles the launches.

Community Focused

Designed for the Linux gaming community with extensibility and customization in mind.

SUPPORTED PLATFORMS

GINUX will be designed to support launching games from the following platforms:

  • Steam - Native Linux games and Proton titles
  • Battle.net - Via Wine with optimized prefixes
  • Standalone Games - Native Linux executables
  • Custom Launchers - Add your own game shortcuts

More platforms coming soon!

TECHNICAL STACK (UNDER DEV)

Architecture: Web-based frontend + Local bridge backend

Protocol: Custom mylauncher:// URL scheme for game launching

Compatibility: Any Linux distribution with bash and xdg-utils

Security: Local execution only - no remote code execution

Development Stage: Active development, beta testing phase
