About Watchdog AI BETA

Watchdog AI BETA, developed by WebXOS, is a cutting-edge front-end tool designed to monitor and optimize the energy efficiency and carbon footprint of web applications. Built with a focus on sustainability, it empowers companies and developers to create eco-friendly software by providing real-time insights into resource usage and environmental impact. Watchdog operates entirely on the client side, ensuring no personal data is harvested, and delivers green energy reports for actionable insights.

How Tests Are Performed

Watchdog AI BETA runs a suite of tests to evaluate the environmental impact of browser-based activities. Each test is designed to measure specific resource usage metrics:

  • CPU Efficiency: Measures CPU performance by executing a computational stress test (e.g., 500,000 iterations of random number calculations). The test records execution time, with faster times indicating lower CPU load and energy use. Scores range from 4 to 8 based on thresholds adjusted for device performance.
  • Memory Usage: Evaluates memory allocation by creating a large array (200,000 elements) and measuring the difference in JavaScript heap size. Lower memory usage (e.g., <2MB) yields higher scores, reflecting efficient resource management and reduced energy consumption.
  • Power Usage: Estimates power consumption based on device performance and test results, calibrated to a baseline of 40W. Scores reflect efficiency, with lower wattage (e.g., <30W) indicating eco-friendly operation.
  • Battery Impact: Assesses battery level using the Battery API (if available), scoring higher for levels above 50%. Lower battery levels increase carbon footprint due to higher charging frequency, making this test critical for mobile devices.

Tests are calibrated for mobile and desktop devices, with reduced iterations on mobile to ensure lightweight operation. Results are aggregated into an Eco Score (0-100) and a Carbon Footprint (kg CO2/h), providing a clear measure of environmental impact.
