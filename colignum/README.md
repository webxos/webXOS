# COLIGNUM

**A modular graphic design & creative studio suite** 

— part of [webXOS](https://github.com/webxos/webXOS), the web-based operating system.

COLIGNUM is an experimental collection of high-fidelity, standalone web tools focused on **visual experimentation, generative art, ASCII culture, 3D, and creative workflows**. Think of it as a digital graphic design agency living entirely in the browser — no install, no servers, maximum immediacy.

## Tools

### Core Experiences

- **`colignum.html`** — Main landing / showcase page with 3D generator, procedural objects, and studio overview.
- **`ascii_studio.html`** — Advanced ASCII art studio  
  - Real-time 3D shape → ASCII conversion  
  - 40+ optical illusion patterns  
  - Image-to-ASCII (JPEG/PNG upload)  
  - Live overlays, glitch effects, export to PNG/MP4
- **`perfect_png.html`** — Node-based visual editor for perfect, high-resolution PNG exports
  - Drag-and-drop nodes, connections, drawing, text labels  
  - Infinite canvas with grid, undo/redo, export powerhouse
- **`point_cloud.html`** — Point cloud & 3D data visualization studio (beta)
- **`blackbox.html`** — Mysterium Blackbox Tesseract: interactive 3D wireframe environment with CLI and face terminals
- **`trashdoctor.html`** — Universal document compiler & IndexedDB archive  
  - Drag & drop *anything* (images, PDFs, scripts, spreadsheets...)  
  - Local storage, ZIP export, metadata tracking

## Features

- **Pure browser-based** — Everything runs client-side (Three.js, WebGL, Canvas, IndexedDB)
- **Monochrome / high-contrast aesthetic** — Retro-futurist, cyber-minimalist design language
- **Export-focused** — PNG (including 8K), MP4 demos, ZIP archives, point cloud data
- **Highly interactive** — Sliders, real-time previews, procedural generation, orbit controls
- **Modular & extensible** — Easy to fork individual tools or integrate into webXOS

## Single File Design

Designed with simplicity and low latency in mind.

## Tech Stack

- HTML5 / CSS3 (custom monochrome themes)
- JavaScript (vanilla + modules)
- Three.js (r128) for 3D
- Canvas / WebGL
- IndexedDB for local persistence (Trashdoctor)
- No build step required
