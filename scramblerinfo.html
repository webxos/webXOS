Getting Started

The Webxos Scrambler allows you to encode messages into unique SCR- codes and decode them back to their original text. It features a sleek, Matrix-inspired interface with animated text effects.

  • Access the App: Open the Scrambler Here.
  • PWA Enhanced: No Installation needed, app runs on complete front end mobile html (no logins or user data).

How to Use

  • Display Text: Enter any text in the input field and click "Execute" to display it with a Matrix-style animation.
  • Scramble Text: After displaying text, click "Scramble" to encode it into an SCR- code (e.g., SCR-12345678). The code is automatically copied to your clipboard.
  • Decode SCR- Code: Enter an SCR- code (e.g., SCR-12345678) and click "Execute" to decode and display the original message.
  • Clear Console: Click "Clear" to reset the input, output, and display.

Features

  • Chromatic Encryption: Messages are encrypted using AES with a key derived from a unique hash, ensuring security.
  • Matrix Animations: Text is displayed with dissolve and rebuild effects, with per-digit coloring for SCR- codes.
  • Persistent Storage: SCR- codes and their data are stored locally using IndexedDB for reliable decoding.
  • Clipboard Integration: SCR- codes are automatically copied for easy sharing.

Troubleshooting

  • Invalid SCR- Code: Ensure the code starts with "SCR-" and is 12 characters long. Codes are case-insensitive.
  • App Not Responding: Wait for animations to complete before clicking buttons, or click "Clear" to reset.
  • Storage Issues: Clear your browser's IndexedDB data if decoding fails consistently.
